# python-programming-exercises

## Level 1 Beginner

初学者指的是刚完成入门 Python 课程的人。他能用一两个 Python 类或函数解决一些问题。通常，答案可以直接在教材中找到。

## Level 2 Intermediate

中级指的是刚学过 Python，但之前已有较强编程基础的人。他应该能够解决可能需要 3 到 3 个 Python 类或函数的问题。答案无法直接在教科书中找到。

## Level 3 Advanced

他应该用 Python 解决更复杂的问题，使用更丰富的库、函数、数据结构和算法。他需要用多个 Python 标准软件包和高级技术来解决这个问题。
